= Phase 1: Foundation — Governed Data and Models
:navtitle: Phase 1 Overview
:toc: macro

**Value:** No Shadow IT. One supply chain for credentials and models.

== Outcome

Associates can connect workbenches and model servers to storage and use a private, curated model catalog.

== Takeaway

* Credentials in Kubernetes Secrets (Data Connections); models in a private Registry; Catalog as the showroom.
* **Engineering outcome:** Reproducible, governed model and data access.

Use the navigation menu to work through the module pages in order: Storage, then Registry, then Catalog.

== Checkpoint

* Data Connection created and attached; PVC bound; model registered and visible in Catalog; deployment from wizard succeeds.
