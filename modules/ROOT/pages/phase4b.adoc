= Phase 4b: Token Factory — Models as a Service
:navtitle: Phase 4b Token Factory MaaS
:toc: macro

**Value:** Scale and sovereignty. From "one model per project" to a governed, API-first token factory.

== Outcome

Associates understand and can operate a MaaS-style platform (Gateway, Kuadrant, llm-d) for multi-tenant, API-first token generation.

== Takeaway

* llm-d is the distributed inference backbone; MaaS adds service tiers, API keys, and governance.
* **Engineering outcome:** Centralized, governed LLM API with quotas and auth.
